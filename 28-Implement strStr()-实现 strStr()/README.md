字串搜索，有两种方法：

- 暴力搜索，即将b与a的每个字串对比，直到找到匹配，或者结束。
- KMP算法。

而KMP算法的主要分为两步：

- 求取模式串的next数组。
- 根据next数组来对模式串进行回溯求解，而不再对文本串回溯。

而第一步又最为关键，其中最重要的点为：当前位置的值`needle[i]`与遍历的值`neelde[j + 1]`不相等时，应当进行将j重置为next数组前一位的值`next[j]`。
